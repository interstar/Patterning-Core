Module Dimensions

Size of the modules (single unit) is 101mm h x 25mm w. Wider modules are a multiple of 25.4mm (1‚Äù) minuses approx. 0.5mm gap tolerance

PCB (single unit) is 100 x 25mm

The standards case allows for a depth of 25mm, plus a 1.5 mm faceplate.

A module is composed of

The front panel (thickness : 1.5mm)
Hex spacers M3 8mm high (female-male) between the front panel and PCB
PCB (thickness: 2mm)
Hex spacers M3 15mm high (female-female)

Panel Template

(def cut-style {:stroke (p-color 255 0 0)})
(def engrave-style {:stroke (p-color 0 0 255)})

(def wide 25.0)
(def high 101.0)
(def screw-y 3.4)
(def screw-x (/ wide 2))
(def cut-rect  (rect 0 0 wide high  cut-style))
(def gap wide)

(defn circ [cx cy r style] (poly cx cy r 30 style))
(def screw-holes 
   (stack
       (circ screw-x screw-y 1.5 cut-style) 
       (circ screw-x (- high screw-y) 1.5 cut-style)
   )
)

;; Define the custom panel

(defn panel [x] 
    (->>
      (stack
       (rect 0 0 wide high cut-style)
       screw-holes
      )
      (translate x 0)
    )
)


;; Don't forget to resize the whole image to 4 * gap mm wide

(->>
  (apply stack (take 4
    (map panel (iterate #(+ % gap) 0))
  ))
  reframe
  
  (stack (rect -1 -1 2 2 cut-style) )
)
